## ğŸš€ Getting Started

### **Prerequisites**
- Web server (Apache, Nginx, or any HTTP server)
- Modern web browser
- PHP support (for contact form functionality)

### **Installation**

1. **Clone the repository**
   ```bash
   git clone https://github.com/sathinathpadhi2001/YMR_ShipChandlers.git
   cd YMR_ShipChandlers
   ```

2. **Deploy to web server**
   - Upload all files to your web server
   - Ensure PHP is enabled for contact form functionality
   - Configure email settings in `contact-handler.php`

3. **Local Development**
   ```bash
   # Using Python (if available)
   python -m http.server 8000
   
   # Using Node.js (if available)
   npx http-server
   
   # Or simply open index.html in your browser
   ```
